# ☑️ Checklists — Orbeey

## Checklist de onboarding de cliente
- [ ] Contrato assinado
- [ ] Acesso às ferramentas concedido
- [ ] Briefing coletado
- [ ] Reunião de kick-off realizada
- [ ] Equipe interna briefada
- [ ] Primeiro entregável definido

## Checklist de offboarding de cliente
- [ ] Relatório final enviado
- [ ] Acessos revogados
- [ ] Documentação arquivada
- [ ] Feedback coletado
- [ ] Nota fiscal emitida

## Checklist semanal da operação
- [ ] Relatórios enviados
- [ ] Tarefas atrasadas verificadas
- [ ] Financeiro conferido
- [ ] NPS atualizado

## Checklist mensal
- [ ] Relatório mensal enviado a todos os clientes
- [ ] MRR calculado
- [ ] Margem por cliente calculada
- [ ] Reunião de alinhamento com equipe
