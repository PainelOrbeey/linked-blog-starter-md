# 🎛️ Comandos da IA — Orbeey Brain

## Comandos de memória
- `@salvar cliente [nome]` → Cria ficha do cliente
- `@atualizar cliente [nome]` → Atualiza dados do cliente
- `@buscar cliente [nome]` → Recupera histórico do cliente

## Comandos de relatório
- `@relatorio semanal [cliente]` → Gera relatório semanal
- `@relatorio mensal [cliente]` → Gera relatório mensal
- `@resumo geral` → Resumo de todos os clientes

## Comandos de análise
- `@analisar churn` → Identifica clientes em risco
- `@analisar financeiro` → Resumo financeiro atual
- `@diagnostico [cliente]` → Diagnóstico completo do cliente

## Comandos de execução
- `@criar tarefa` → Cria nova tarefa
- `@cobrar [responsável]` → Envia lembrete de tarefa
- `@alerta [urgente]` → Dispara alerta para o time
