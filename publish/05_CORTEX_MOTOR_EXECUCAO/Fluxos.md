# 🔄 Fluxos de Trabalho — Orbeey

## Fluxo de nova demanda
1. Cliente solicita
2. Gestor avalia escopo
3. Gestor atribui responsável
4. Responsável confirma prazo
5. Execução
6. Revisão interna
7. Envio para aprovação do cliente
8. Ajustes (se necessário)
9. Entrega final

## Fluxo de novo cliente
1. Lead fechado pelo comercial
2. Contrato enviado
3. Contrato assinado
4. Onboarding iniciado
5. Kick-off realizado
6. Operação iniciada

## Fluxo de relatório
1. Coleta de dados
2. Análise
3. Geração do relatório
4. Revisão
5. Envio ao cliente
