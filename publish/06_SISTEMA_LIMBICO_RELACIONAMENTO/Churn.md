# 🚨 Churn — Monitoramento

## Clientes em risco
| Cliente | Risco | Sinais | Ação em andamento | Responsável |
|---------|-------|--------|------------------|-------------|
|         | Alto/Médio/Baixo | | | |

## Sinais de churn
- 3+ reclamações no mês
- NPS abaixo de 6
- Atraso no pagamento
- Redução de demandas
- Falta de resposta
- Solicitação de "pausa"

## Protocolo de retenção
1. Identificar sinal
2. Escalar para gestor
3. Contato em até 24h
4. Reunião de alinhamento
5. Plano de ação
6. Acompanhamento semanal
