# 🏢 Rotina da Agência

## Rotina diária
- [ ] Verificar tarefas atrasadas
- [ ] Checar mensagens de clientes
- [ ] Atualizar status das entregas
- [ ] Registrar novas demandas

## Rotina semanal (segunda-feira)
- [ ] Gerar relatórios semanais de todos os clientes
- [ ] Reunião de alinhamento de equipe
- [ ] Revisar prioridades da semana
- [ ] Checar prazos críticos

## Rotina mensal (dia 1)
- [ ] Relatório mensal de todos os clientes
- [ ] Revisão de MRR
- [ ] Revisão de margem por cliente
- [ ] Planejamento do mês
