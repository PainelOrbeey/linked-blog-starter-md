# 🚨 Protocolo de Emergência

## Situações de emergência
- Cliente ameaça processar
- Vazamento de dados
- Erro grave em campanha
- Colapso de sistema crítico
- Perda de acesso a ferramentas

## Protocolo geral
1. Parar a operação afetada imediatamente
2. Comunicar gestor em até 15 min
3. Identificar causa raiz
4. Comunicar cliente com transparência
5. Acionar responsável técnico/jurídico
6. Documentar ocorrência
7. Implementar correção
8. Relatório pós-incidente
