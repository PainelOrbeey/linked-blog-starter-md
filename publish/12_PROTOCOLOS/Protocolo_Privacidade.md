# 🔒 Protocolo de Privacidade — LGPD

## Dados protegidos
- CPF/CNPJ dos clientes
- Dados de pagamento
- Estratégias confidenciais
- Contratos

## Regras de acesso
- Dados de clientes: somente equipe interna autorizada
- Dados financeiros: somente gestores
- Contratos: somente jurídico + gestores

## Conformidade LGPD
- Coleta de dados apenas com consentimento
- Dados armazenados com segurança
- Direito de exclusão respeitado

## Em caso de vazamento
1. Identificar escopo
2. Comunicar gestão em até 2h
3. Acionar jurídico
4. Comunicar clientes afetados
5. Registrar ocorrência
