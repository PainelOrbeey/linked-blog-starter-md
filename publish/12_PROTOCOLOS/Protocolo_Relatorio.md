# 📊 Protocolo de Relatório

## Padrão de qualidade
- [ ] Dados atualizados
- [ ] Sem erros de cálculo
- [ ] Insights claros (não só números)
- [ ] Próximos passos definidos
- [ ] Visual limpo e profissional
- [ ] Linguagem acessível ao cliente

## Estrutura obrigatória
1. Resumo executivo (3 linhas)
2. Resultados do período
3. O que funcionou
4. O que precisa melhorar
5. Próximos passos

## Aprovação antes de enviar
- Relatório mensal → revisão do gestor antes de enviar
- Relatório semanal → envio direto pelo responsável
